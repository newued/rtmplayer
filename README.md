### 一个rtmp视频播放器(flash版本)
技术框架：actionscript3 html css javascript

## 功能描述
1.rtmp格式视频播放(点播，直播)<br>
2.提供外部js脚本设置弹幕，打点，弹幕设置，开关灯，播放暂停控制<br>
3.支持缩略图<br>
4.支持断点续播<br>
5.全屏切换<br>
